## 📋 项目概述

### 项目名称
**Miks Valorant Agent** - Valorant 特工信息查询站

### 产品定位
一个专注于《Valorant》特工信息的查询网站，提供全面的特工技能详解、对比工具和使用攻略，帮助玩家快速了解特工特点，选择合适的特工进行游戏。

### 目标用户
- Valorant 新手玩家（快速了解特工）
- 进阶玩家（查询特工克制关系）
- 内容创作者（获取特工资料）
- 电竞爱好者（了解特工数据）

### 核心价值
- ✅ **快速查询** - 3 秒内找到任意特工信息
- ✅ **全面准确** - 官方 API 数据，实时更新
- ✅ **直观对比** - 多特工技能对比功能
- ✅ **中文友好** - 完整中文翻译，新手友好

## 🎯 MVP 功能规划

### 核心功能（P0 - 必须上线）

#### 1. 特工列表页

| 功能点 | 说明 | 优先级 |
|--------|------|--------|
| 特工卡片展示 | 头像、名称、定位（决斗/先锋/控场/哨卫） | P0 |
| 筛选功能 | 按定位筛选（决斗者/先锋/控场者/哨卫） | P0 |
| 搜索功能 | 支持特工名称搜索 | P0 |
| 点击跳转 | 点击卡片进入特工详情页 | P0 |

#### 2. 特工详情页

| 功能点 | 说明 | 优先级 |
|--------|------|--------|
| 特工基本信息 | 头像、名称、定位、背景简介 | P0 |
| 技能展示 | 4 个技能（C/Q/E/X）图标、名称、描述、冷却时间 | P0 |
| 技能图片/GIF | 每个技能配图或动图 | P0 |
| 返回按钮 | 返回列表页 | P0 |

#### 3. 数据源集成

| 功能点 | 说明 | 优先级 |
|--------|------|--------|
| Valorant API | 集成 https://valorant-api.com/ | P0 |
| 数据缓存 | 减少 API 调用，提升加载速度 | P0 |
| 错误处理 | API 失败时显示友好提示 | P0 |

---

### 次要功能（P1 - 第一版迭代）

#### 4. 特工对比功能

| 功能点 | 说明 | 优先级 |

| 选择对比 | 选择 2-3 个特工进行对比 | P1 |

#### 5. 特工攻略

| 功能点 | 说明 | 优先级 |
|--------|------|--------|

| 地图推荐 | 适合的地图 | P1 |

## 🔧 技术方案

### 技术栈

| 层级 | 技术 | 说明 |
|------|------|------|
| **前端框架** | Next.js 14 | React 全栈框架，SEO 友好 |
| **样式库** | Tailwind CSS | 快速开发，响应式设计 |
| **动画库** | Framer Motion | 特工技能动画效果 |
| **图标库** | Lucide React | 简洁图标 |
| **部署平台** | Vercel | 免费、自动部署、全球 CDN |
| **数据源** | Valorant API | https://valorant-api.com/ |

### API 接口

**Valorant API 端点：**

```typescript
// 获取所有特工
GET https://valorant-api.com/v1/agents

// 获取特工详情（含技能）
GET https://valorant-api.com/v1/agents/{uuid}?language=zh-CN

// 获取特工图片
GET https://valorant-api.com/v1/agents/{uuid}/displayicon
```

**数据示例：**

```json
{
  "status": 200,
  "data": {
    "uuid": "agent-uuid",
    "displayName": "Jett",
    "description": "来自日本的决斗者...",
    "developerName": "Jett",
    "role": {
      "displayName": "Duelist",
      "displayIcon": "icon-url"
    },
    "abilities": [
      {
        "slot": "C",
        "displayName": "Cloud Burst",
        "description": "投掷云爆弹...",
        "displayIcon": "icon-url"
      }
    ]
  }
}
```

## ⚠️ 风险与应对

| 风险 | 影响 | 应对方案 |
|------|------|----------|
| API 不稳定 | 数据加载失败 | 增加数据缓存、错误提示 |
| 数据更新延迟 | 新特工上线慢 | 监控 API 更新、手动触发刷新 |
| 版权问题 | 图片/内容侵权 | 使用官方 API、注明来源 |
| 流量过大 | 服务器压力 | Vercel 自动扩容、CDN 加速 |
